# Testing

## Automated tests

Run from the repository root:

```bash
# Run all tests
npm test

# Unit tests only
npm run test:unit

# Integration tests only
npm run test:integration

# Watch mode
npm run test:watch
```

## Manual verification

Generate sample output videos to visually verify rendering behavior:

```bash
# Run all demos
node examples/run-examples.js

# Run specific demos by name (partial match)
node examples/run-examples.js transitions
node examples/run-examples.js torture ken
```

Output goes to `examples/output/`. Each demo script includes a `WHAT TO CHECK` section describing the expected visual result at every timestamp.
